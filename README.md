# Dotfiles

A collection of configuration files with a one-click install.

## Setup

To install the dependencies for the configuration run `bash ./install_deps.sh`.

In order to install the symlinks run `bash ./setup.sh`

## i3 WM requirements

- i3-gaps
- polybar
- kitty
- alacritty
- dmenu
- rofi
- picom

## Neovim Requirements

- Packer
- RipGrep
- fzf
